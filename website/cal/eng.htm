Calendar Description

The English Hybrid calendar is a hybrid variant combining the Julian Annunciation, the Julian and the Gregorian calendars.

Use Case

  • England and Wales from the 12th centry.
  • Former English overseas colonies and territories, from the 12th century to the current day.

Definition

Record
schemeyearmonthdayunshift

The Record consists of four Fields named scheme, year, month, and day and a read only Field unshift. The unshifted field holds the historical year as calculated by the underlying Julian calendar.

As a Hybrid calendar, it has no single epoch.

Script

Default Script
vocab sch { name Scheme names; lang en; style-name Scheme Sch; tokens { 0 "Old Style" OS; 1 Julian; 2 "New Style" NS; }; }; grammar hy { alias field { Day day; Month month; Year year; Hist-Year unshift; Scheme scheme; }; alias format-number-code { Day dd; Month mm; Year yyyy; Hist-Year y; Scheme s; }; vocabs m sch; format pref @(Day) @(Month:m.a) @(Year/Hist-Year); format @(Day) @(Month:m.a) @(Year); format @(Day) @(Month:m.a) @(Year) @(Scheme:sch.a); }; scheme eng {name English Hybrid; hybrid { fields year month day unshift; scheme j325; scheme j {begin 2360975;}; scheme g {begin 2361222;}; }; grammar hy; };

The script makes use of the globally available read-only field wday to output weekdays.

The hy grammar uses the m (Julian months) and w (Weekdays) vocabs as defined for the Julian calendar.
