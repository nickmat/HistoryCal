Calendar Description

The 7 day week is a cyclic calendar with no major count and so can only be used in conjunction with other calendars.

Use Case

Thought to have originated in ancient Babylonia around 600 BCE. It is used with the Julian, Gregorian, Hebrew and Islamic and other calendars.

There are more than one way of numbering the week days. For convenience, the library has chosen Monday as the first day. Saturday and Sunday are alternative start days.

Record
wday

The Record consists of a single extended Field named wday. The Epoch is wday 1 = day 1 (jdn) a Monday.

Script

Default Script
vocab "w" { name "Weekday names"; lang "en"; stylename "Weekday", "WDay"; tokens { 1, "Monday", "Mon"; 2, "Tuesday", "Tue"; 3, "Wednesday", "Wed"; 4, "Thursday", "Thur"; 5, "Friday", "Fri"; 6, "Saturday", "Sat"; 7, "Sunday", "Sun"; } }

Since this calendar is only used in conjunction with others, only the vocab associated with it is shown.

The script has no dependencies.
