Introduction

HistoryCal is a calendar calculator that can perform a number of tasks. The simplest being the conversion of dates from one calendar scheme to another. It can handle ranges of dates and carry out certain operations, such as returning the starting range given a fixed date and an age in years or months.

The calculator can be extended using a built-in script language. This can be used to create variations on the basic calendar schemes, such as shifting the new year to a different point in the calendar, or creating regnal calendars. The same script can be used to describe the grammar, formatting and vocabularies used by the calendars.

Inputting Dates

To input a date, simply select the calendar scheme you are using and then the format to be used. The format selected will determine the order of the various date elements and (in future) how punctuation is interpreted. Then use the single text box to enter the date. You can enter the date as all numbers separated by spaces. For elements that have names, you may enter the name, in full or abbreviated. You can either type the name or select it from the text lists.

The following rules apply to entering dates:-

  • Numbers may start with a minus sign or hyphen (-). Used for some proleptic calendars.
  • Numbers are separated by a space ( ), comer (,) or colon (:).
  • Text names (which may contain spaces) are separated by a comer (,) or colon (:).
  • Numbers and text do not need separators but may use a space ( ), comer (,) or colon (:).
  • Double dating may be used in some circumstances by using a slash (/) between the numbers.
  • Future versions may provide custom variations on these rules, for instance, to allow ISO format (with hyphens).

Inputting Ranges

A range is a set of continuous days and described by the inclusive first and last days in the set. A single date may be considered as a range of one day.

Ranges can be entered by entering two dates separated by a tilde (~). It is also possible to enter a shorthand range by omitting the lesser significant elements of a date, so, for a day month year type calendar, just entering the month and year or just a year is the equivalent of entering a range. For example, "Sep 1948" is equivalent to "1 Sep 1948 ~ 30 Sep 1948" and "1948" to "1 Jan 1948 ~ 31 Dec 1948".

Multiple ranges may be entered by separating them with the vertical bar (|) character.

Setting the Output

The output is obtained by selecting the calendar scheme and the format to be used. Hopefully the format specification is self explanatory. In general, lower case letters indicate numbers are output, element names with an initial upper case letter indicate the text names are output. If the element name is abbreviated then the text name will be as well.

Optional Output

The View menu may be used to set the following options.

Show intermediate values. These are the Julian Day Numbers (jdn) that are used in the conversion. This is useful if you have reason to believe the conversion is wrong. The intermediate values can be checked to see if the error is on the input side or the output side.

Include range lengths. When a range, or ranges, are output, the total number of days is appended.
