Introduction

HistoryCal is an program to convert between various current and historical calendars. The input and output calendars may be variations of a common basic calendar and may use different formats to describe them.

As well as converting single days, the program can work with ranges of days. A range can in shorthand as, say, a year or a year and month. Not all date formats can take advantage of this ability.

HistoryCal can combine ranges using mathematical set operations. The values 'past' and 'future' can be used so expressions such as "on or before" can be represented as the range 'past..1956'.

Other calculations can be carried out, such as calculating an age on a given day or, given an age and date, calculate the date range the birth date falls.

The entry cal use a mixture of calendar schemes and formats using a date 'signature'. For example, the Gregorian calendar scheme written using the "Month Day, Year" format can prefix the date with 'g:mdy#' (without the '').
