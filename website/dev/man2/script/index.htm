The Glich Script Language

Although the main purpose of Glich is to be extended for other specialised projects, it is a fully functional general purpose and self contained language in its own right.

This manual describes the language and then goes on to describe the Hics extension. A tutorial on Glich and Hics using the Command Line console program is included.
