HistoryCal version 0.0.1 alpha

This is a early alpha release to allow anybody interested in the project to follow its progress.

A working Windows (XT, Vista, 7, 8) version is available as file hcal_0-0-1_setup.exe

The source code is available in files hcal_0-0-1_src.zip - DOS/Win line endings. These source files have been successfully compiled on Windows 8 using Visual Studio C++ 2008 Express.

Also available are the source code files hcal_0-0-1-1_src.gz with Unix line endings. These have been successfully compiled on gnu/linux Mint.

See the included build/build.txt file for a description of the build proccess.

A Mac OSX version is still being worked on.
