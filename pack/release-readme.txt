HistoryCal version 0.0.1 alpha

This is a early alpha release to allow anybody interested in the project to follow its progress.

A working Windows (XT, Vista, 7, 8) version is available as file hcal_0-0-1_setup.exe

The source code is available in files hcal_0-0-1_src.zip. Unix and Mac OSX versions are still being worked on and the source code for these platforms will be released shortly. See the included build/build.txt file for a description of the build proccess.
