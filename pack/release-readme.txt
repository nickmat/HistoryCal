HistoryCal version 0.0.2 alpha

The alpha releases are early versions to allow anybody interested in the project to follow its progress. This version adds the "today" input keyword and corrects some spelling errors.

A working Windows (XT, Vista, 7, 8) version is available as file hcal_0-0-2_setup.exe

The source code is available in files  hcal_0-0-2_src.zip (DOS/Win line endings) and  hcal_0-0-2_src.gz (Unix line endings). The source files have been successfully compiled on gnu/linux Ubuntu (Mint) and on Mac OS X (Yosemite). See the included build/build.txt file for a description of the build proccess for linux. The Mac OS X version is still being worked on.
